# Skillbound

A RuneScape-inspired, turn-based browser idle/skilling game. Train 15 skills from level 1 to 99, gather resources, smith gear, cook food, brew potions, and take down 5 bosses for unique drops.

## Features

- **15 skills** with the classic exponential 1–99 XP curve: Attack, Strength, Defence, Hitpoints, Ranged, Magic, Woodcutting, Mining, Fishing, Foraging, Smithing, Cooking, Crafting, Fletching, Alchemy
- **Turn-based combat** — accuracy and max-hit rolls driven by your stats and equipment; eat food mid-fight (the enemy gets a free swing), drink potions for boosts, or flee
- **Interlocking economy** — ores → bars → gear, logs + flax → bows, fish → food, herbs → potions, hides → leather armour, gems → amulets
- **6 equipment tiers** (bronze → rune) across melee, ranged, and magic styles
- **5 bosses** gated by combat level, each with unique drops
- **Local-first** — progress autosaves to localStorage

## Run it

```sh
npm install
npm run dev
```
